# 🤖 Vikas AI

Vikas AI is a Python based personal assistant project.

It can understand user commands and perform tasks like file management, notes, history and memory handling.

## 🚀 Features

### 📂 File Management
- Create file
- Write file
- Read file
- Append file
- Delete file
- Rename file
- Search file
- Copy file
- Move file
- File information
- Count files

### 📁 Folder Management
- Create folder
- List folders
- Delete folder

### 🧠 Assistant Features
- Notes
- History
- Memory
- Natural language command handling

## 🛠️ Technology Used

- Python
- JSON
- File System Handling
- Modular Architecture
